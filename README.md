# Draugiem.lv profila jaunumi #

**Draugiem.lv profila jaunumi** ir Google Chrome paplašinājums, kas paziņojumu veidā attēlo jaunumus (vēstules, galerijas, statistika utml.) lietotāja Draugiem.lv profilā.
